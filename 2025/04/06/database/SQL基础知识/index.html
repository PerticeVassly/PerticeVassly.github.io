SQL分类

DDL：数据定义语言，创建和修改数据库对象的语言。DDL语句包括CREATE、ALTER、DROP等

DML：数据操作语言，操作数据库中数据的语言。DML语句包括INSERT、UPDATE、DELETE等

TCL：事务控制语言，控制数据库事务的语言。TCL语句包括COMMIT、ROLLBACK、SAVEPOINT等

DCL：数据控制语言，控制数据库访问权限的语言。DCL语句包括GRANT、REVOKE等

一些关键词

LIKE

LIKE：在where子句中使用，只有字段是文本值的时候才能使用LIKE，支持两个通配符 %和_，%表示任意长度的字符串，_表示一个字符

JOIN

JOIN: 在每个表中选择一个字段，并且对这些字段的值进行比较，值相同的两条记录合并为一条。链接表的本质就是将不同表的记录合并起来形成一张新表。此表知识临时的，仅存在于查询中。如果两张表的关联字段相同可以使用USING关键字来简化语句。inner join Orders o using (OrderID)

INNER JOIN：内连接，返回两个表中匹配的记录

LEFT JOIN：左连接，返回左表中所有的记录和右表中匹配的记录，如果右表中没有匹配的记录，则返回NULL

RIGHT JOIN：右连接，返回右表中所有的记录和左表中匹配的记录，如果左表中没有匹配的记录，则返回NULL

FULL JOIN：全连接，返回两个表中所有的记录，如果没有匹配的记录，则返回NULL

SELF JOIN：自连接，表与表之间的连接，表与自身的连接

CROSS JOIN：笛卡尔积，返回两个表中所有的记录的组合

DELIMITER

当你创建存储过程时，MySql会将分号作为语句的结束符，因此在创建存储过程时需要使用DELIMITER命令来改变分隔符。使得你的过程中可以使用分号。

事务处理

不能回退SELECT语句, 回退SELECT语句没有意义；也不能回退CREATE和DROP语句

MySql默认是隐式提交，美提交一条语句就把这条语句当作一个事务然后提交。当出现START TRANSACTION语句时，MySql会将隐式提交关闭，直到出现COMMIT或ROLLBACK语句时才会重新打开隐式提交。

通过 set autocommit=0 可以取消自动提交，直到 set autocommit=1 才会提交；autocommit 标记是针对每个连接而不是针对服务器的。

游标

cursor 是一个存储在DBMS中的数据库查询，不是一条SELECT语句。而是被该语句检索出来的结果集。

在存储过程中可以使用游标来遍历结果集。

但 MySQL 不支持为特定游标声明专属的 handler（不像 PostgreSQL 有 block-scope），所以你需要：拆成多个独立的 BEGIN…END 块.

触发器

触发器是一种与表操作有关的数据库对象，当触发器所在表上出现指定事件时，将调用该对象，即表的操作事件触发表上的触发器的执行。

MySQL 不允许在触发器中使用 CALL 语句 ，也就是不能调用存储过程。

MySQL 中定义了 NEW 和 OLD 关键字，用来表示触发器的所在表中，触发了触发器的那一行数据。
