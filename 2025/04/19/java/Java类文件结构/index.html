背景

Clojure Groovy Kotlin Scala JRuby Jython等语言都可以编译成Java字节码，运行在JVM上。

一个class文件主要包括文件标志，小版本号，大版本号，常量池的数量，常量池，访问标记，当前类，父类，接口数量，接口，字段数量，字段，方法数量，方法，属性数量，属性。

Magic Number

U4 0xCAFEBABE 每个class文件的头四个文件称之为magic number，表示该文件是一个class文件。

版本号（Minor&Major Version）

U2 minor_version // Class的小版本号
U2 major_version // Class的大版本号

高版本的Java虚拟机可以执行低版本的Class文件，低版本的Java虚拟机不能执行高版本编译生成的Class文件

常量池 （Const Pool）

常量包含：

类和接口的全限定名

字段的名称和表述符

方法的名称和描述符

访问标志 （Access Flags）

用于标识一些类或者接口层次的访问信息，包括：这个Class是类还是接口 ，是否为public或者abstract类型，如果是类的话是否是final等等

当前类（This Class）父类 （Super Class）接口（Interfaces）

所有的类都继承自Object类，接口可以多继承。

字段表（Fields）

其中的name_index和descriptor_index分别指向常量池中的字段名称和描述符。

方法表（Methods）
