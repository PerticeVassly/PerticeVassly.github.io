mysqlloadCsv问题

环境

mysql Ver 8.0.41-0ubuntu0.22.04.1 for Linux on x86_64 ((Ubuntu)) （wsl2）

我直接在wsl2中使用mysql-workbench连接mysql（本地），对应mysql-workbench版本为 MySQL Workbench CE (GPL) 8.0.32 CE build 2612062

实现导入csv

1. 首先要配置mysql的secure_file_priv参数

这个参数限制了load data infile的路径

wsl2下是到/etc/mysql/mysql.conf.d/mysqld.cnf中修改（尽管会带来安全问题）

参考：https://superuser.com/questions/1088512/how-to-disable-secure-file-priv-mysql-ubuntu

2. load时候需要指定local关键字

3. 需要指定local-infile=1

此处需要同时在mysql服务器和客户端两端都设置

对于服务器端，可以在/etc/mysql/mysql.conf.d/mysqld.cnf中添加
