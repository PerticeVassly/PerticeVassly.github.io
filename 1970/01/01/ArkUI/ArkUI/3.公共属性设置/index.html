3.公共属性设置

参考资料：https://www.arkui.club/chapter3/

尺寸设置

width/height

设置组件大小的时候支持number和string类型

number类型的时候，单位默认为vp

string类型的时候，需要带上单位，如果不带单位，会默认为vp

当组件同时设置size和width/height的时候，size会覆盖width/height

若子组件的宽和高大于父组件的宽和搞，默认情况下子组件会绘制出父组件的可视范围，此时需要设置clip方法为true，这样子组件会被裁剪掉超出父组件的部分

apectRatio

一般用于设备适配

layoutWeight

设置组件的布局权重，仅仅再Row Column Flex布局中生效，表示父容器在主轴方向上的尺寸按照权重比进行分配，默认值为0

ConstraintSizeOptions

其优先级好于width/height，但是minWidth高于minHeight

边距设置

padding

若设置的是百分比，上下左右内外距都以父容器的width作为基础值

margin

位置设置

绝对位置

设置当前组件在父组件中的位置，参照点在父容器顶点位置

相对定位

设置元素内容的对齐方式，只有当设置的width和height大小超过元素本身的内容大小时才会生效

锚点设置

设置元素在位置定位时候的锚点，以自身顶点位置作为基准点进行偏移。设置该属性，不影响父容器的布局，仅在绘制的时候进行位置调整

提示设置

给当前组件绑定一个菜单项，用户点击组件后弹出菜单

bindContextMenu

content: 自定义菜单项

responseType: 弹窗的响应类型 LongPress 长按触发 RightClick 右键触发

bindPopop

show: 是否显示弹窗

popup: 创建PopupOptions或者CustomPopupOptions

message: 显示的信息提示

placementOnTop: 是否显示在组件上方

primaryButton: 第一个按钮的配置

secondaryButton: 第二个按钮的配置

onStateChange: Popup可见性变化的回调

builder：自定义 Popup 。

placement： Popup 的相对位置。

maskColor： Popup 蒙层颜色。

popupColor： Popup 背景颜色。

enableArrow： Popup 是否显示箭头。

autoCancel： Popup 是否自动隐藏。

onStateChange： Popup 可见性变化的回调。

对齐方式

对齐方式

align 只有当设置的width和height大小超过元素本身的内容大小时才会生效

布局方向

Direction定义了三种布局方式

Ltr: 从左到右

Rtl: 从右到左

Auto : 系统默认布局方向

约束条件

设置子组件在父组件RelativeContainer中的对齐方式，分为水平对齐规则和竖直对齐规则

背景色设置

边框设置

显隐设置

显隐设置

优先级设置

设置当前组件在布局容器中显示的优先级，当父容器空间不足的时候，低优先级的组件会被隐藏

多态设置

显示组件在不同的状态下的显示样式

normal：设置组件默认情况下的显示样式。

pressed：设置组件按下时的显示样式。

disabled：设置组件不可用时的显示样式。

focused：设置组件获取焦点时的显示样式。

clicked：设置组件点击时的显示样式。

@Styles 样式设置

作用是提取组件的公共样式，方便其他组件复用样式，可以定义在组件内部或者组件外部吗，当定义在组件外部的时候需要添加function关键字

@ExtendStyles 样式继承

UI构建中如果组件设置的属性都是相同的可以使用@Extend对于Text进行扩展，提取相同的属性部分

@Extend 装饰器不能定义在 struct 内部，暂时无法在其它页面引入 Extend 样式

渐变设置

angle: 渐变角度，单位为度，取值范围为0-360

direction: 渐变方向

colors: 渐变颜色数组

repeating：是否重复渲染。

形状剪切

遮罩设置

mask: 给当前组件添加指定形状的这招，遮罩的参考系为当前组件的左上角坐标，为了使遮罩显示必须调用fill（）方法为遮罩增加颜色

Circle：添加圆形遮罩

boolean: 是否按照边缘轮廓进行裁剪
