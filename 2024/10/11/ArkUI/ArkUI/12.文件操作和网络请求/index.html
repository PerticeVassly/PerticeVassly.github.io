12.文件操作和网络请求

参考资料：https://www.arkui.club/chapter12/

文件操作

获取App文件目录

@ohos.ability.featureAbility

打开文件

@ohos.fileio

path: 为绝对路径

flags：文件打开方式， 默认0

code description
0o0 只读
0o1 只写
0o2 读写
0o100 创建文件，如果文件不存在（此时mode存在）
0o200 如果追加了 0o100 选项，且文件已经存在，则出错。
0o1000 如果文件存在且以只写或读写的方式打开文件，则将其长度裁剪为零。
0o2000 以追加方式打开，后续写将追加到文件末尾。

mode：文件权限，默认0o666 同linux文件权限 rwxrwxrwx

网络请求

发送http请求需要申请ohos.permission.INTERNET 权限。

发起 http 网络请求限定并发个数为 100 ，超过这一限制的后续请求会失败。

默认支持https，如果要支持http，需要在config.json中配置增加network标签 属性标识 "cleartextTraffic": true。

http请求

@kit.NetworkKit

默认采用get方法 + header : application/json
