5. Longest Palindromic Substring

参考：

https://leetcode.com/problems/longest-palindromic-substring/solutions/4212564/beats-96-49-5-different-approaches-brute-force-eac-dp-ma-recursion/?envType=study-plan-v2&envId=top-100-liked

Given a string s, return the longest palindromic substring in s.

最朴素的想法是遍历所有的subString，判断是否是回文串，复杂度O(n^3)

但是实际上遍历过程重复检验了一些字串很多次。因此考虑是否能通过已有的字符串推断出其他的字符串是否是回文串。

首先基础条件是一个单独的char是回文串，两个相同的char也是回文串。由于一个回文串一定有一个中心（空或者是一个char），那么可以依据这个中心推断出所有的回文串。复杂度O(n^2)

实际上是遍历了所有可能的中心并且向外expand了所有可能的字符串。Beats 58%

或者使用dp方法

O(n^2)的复杂度，Beats 34%

或者

Manacher算法，提供了O(n)的复杂度，具体参考manacher算法一文,(但是最后竟然只beat了 25%)
