Linux概念和历史

A free Unix-type operating system developed under the GNU General Public License.

Unix - Minix - Linux (Linus Torvalds)

Linux分区理论

对于intel机器一定有分区

最多可以有四个主分区

一个主分区可以在其中扩展为多个逻辑分区

MBR - Master Boot RecorD 512 Byte

其地址在Bios中烧录，包括

  • 446 Byte - 引导程序 to boot OS kernel

  • 64 Byte - 分区表，最多四个分区

  • 2 Byte - Magic Number

文件系统理论

包括VFS，EXT2,EXT3,FAT32等等

磁盘划分

最少要创建: /， 750MB（1.5G推荐）;Swap， 物理内存（2G推荐）

推荐创建: /boot， 16MB

其他：/usr /usr/local /var /tmp /opt /home

开机流程

Bios

Checks memory and hardware (POST)

Loads options from non-volatile memory

Checks for boot devices

Load MBR of boot device and executes it

Boot Loader

MBR加载一个Boot Loader到内从中去

A boot loader loads and starts the Linux kernel

Common Boot loaders: LILO, GRUB

Generally configured in /dev/hda, unless other boot loader is used.

多用户多任务

$ 表示是一个普通用户

# 表示是一个root

Linux文件

结构

byte stream

类型

  • regular file - 普通文件

  • character special file - 字符设备文件

  • block special file - 块设备文件

  • socket file - 套接字文件

  • symbolic link - 符号链接文件

  • directory file - 目录文件

权限

三层level

User - 用户创建这个文件

Group - 用户组

Other - 其他用户

三种权限

Read - 读

Write - 写

Execute - 执行

Shell

用户和操作系统之间的接口

作为核外程序而存在

三种加载方式

  • 强制使用bash来运行；实际上是当前shell又开了一个子进程运行了这个程序

  • ./a.sh 上面

  • source a.sh 或者 . a.sh 直接在当前shell中运行
