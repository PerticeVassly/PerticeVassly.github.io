Java 9

模块化系统

模块化系统，任意一个jar文件，只要加上一个模块描述文件（module-info.java）就可以升级为一个模块

jdk被重组为94个模块，通过jlink实现轻量级JRE（只使用必要的模块）

使用exports关键词控制哪些类可以对外开放使用，哪些类只能内部使用

G1成为默认垃圾回收器

快速创建不可见集合

List.of, Set.of, Map.of, Map.ofEntries

不能进行添加、删除、替换、 排序等操作，不然会报 java.lang.UnsupportedOperationException 异常。

实际过程中慎用

String 存储结构优化

char[] 改为 byte[]

接口中允许定义私有方法

也是对于default的一种适配

抽象类和interface之间的界限再次模糊了

try-with-resources增强

可以在其中使用effectively-final变量

Stream&Optional增强

Stream 中增加了新的方法 ofNullable()、dropWhile()、takeWhile() 以及 iterate() 方法的重载方法。

进程API

新增java.lang.ProcessHandle

Concurrent响应流

变量句柄

变量句柄是一个变量或一组变量的引用，包括静态域，非静态域，数组元素和堆外数据结构中的组成部分等。

Java 10

var 关键字

var 关键字只能用于带有构造器的局部变量（包括包装类）和 for 循环中

集合增强

List，Set，Map等集合类增加了copyOf方法，创建对应的不可变集合

java.util.stream.Collectors增加了toUnmodifiableList、toUnmodifiableSet、toUnmodifiableMap方法，创建不可变集合

Optional增强

新增orElseThrow()方法，允许抛出异常

慎用

扩展CDS

引入了类数据共享机制简称CDS，将一组类预处理为共享归档文件，运行时减少内存映射减少Java类的启动时间，多个虚拟机共用同的归档文件，提高资源利用率

Graal JIT编译器

是AOT的基础

Java 11

HTTP Client标准化

使用CompleteableFuture提供非阻塞请求和响应语义

String增强

是否为空，去掉首尾空格，重复字符串，转换为String数组，范围由行终止符分割的字符串集合

Optional新增isEmpty

lambda表达式支持var

Java 12&13

String

新增字符串缩进方法text.indent(int n)，可以在字符串前面添加n个空格

transform方法，允许使用Function接口对字符串进行转换

Files增强

支持比较两个文件的内容，返回第一个不匹配字符的未知，如果相同返回-1L

复杂的数字格式化工具

支持输出1k这样的复杂格式

增强Switch

可以不用写break

instanceof增强

SocketAPI重构

FileSystems

新增将文件内容视为一个文件系统的API

动态CDS存档

字符块

以及对应的formatted，stripIndent, translateEscapes方法

增强switch

java 14&15

更加精准的空指针问题

switch增强yield转正

record关键字预览

文本块特性预览（加入新的转义字符）

TreeMap增强

新增putIfAbsent，computeIfAbsent，computeIfPresent，compute，merge方法

ZGC转正

新增EdDSA数字签名算法

文本块转正

密封类预览

final字段直接把所有继承和实现都堵死了

密封类可以堆继承或者实现它的类进行限制，这个类只能被指定的类继承

Java 16

提出向量api

jdk源代码中启用cpp14特性

启用弹性元空间

instanceof增强转正

Java 17

增强的伪随机数生成器

Random，ThreadLoaclRandom，SplittableRandom

switch的类型匹配 预览

移除实验性的提前AOT编译器和JIT编译器

Java 18

Java 19

外部函数和内存API

可以通过此API和java运行时职位的代码和数据进行互操作，通过高效调用外部函数安全的访问外部内存

逐步抛弃unsafe

虚拟线程特性预览

多个虚拟线程共享一个操作系统线程

Java 20

作用域值

Java 21

字符串模板预览

仿python的字符串模板

序列化集合SequencedCollections

分代ZGC

Switch模式匹配增强

未命名的模式匹配

未命名类和实例main方法

Java 22

instanceof增强，switch增强，新增对于所有类型的模式匹配

javadoc新增markdown支持

准备启用Unsafe中的内存方法

模块导入声明预览

直接使用import module java.base，一键导入常用核心包

灵活的构造函数体

Java要求在构造函数中super(),this()必须是第一行，java22允许在此之前编写语句，只要这些语句不引用正在构造的实例即可
